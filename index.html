Linux Code

Update

  • sudo apt update

install app

  • sudo apt install terminator

root mode

  • sudo su

Reset login user password

  • restart
  • when you see 'GRUB' menu, press the 'e' key
  • scroll down when line begins with 'linux'
  • bring the cursor to 'ro' and change it 'ro' to 'rw'
  • bring cursor forward to 'quiet splash'and delete remaining words and type 'quiet splash inti=/bin/bash'
  • Press Ctrl+x or F10 to boot
  • In Console change the user password
  • passwd 'username'
  • poweroff -f

Test Connection

  • ping -c 4 8.8.8.8
  • ping -c4 127.0.0.1

Show all child items

  • ls -lha /etc
  • ls -la --human-readable /etc

Show information about file system size in GB

  • df -h
  • df -h /dev

Get help for commands

  • ls --help
  • man ls
      Inside man page
    • press 'h' to get more help
    • to search press '/' and type word to search
      • 'n' to next search
      • 'shift+n' to previous search
    • 'shift+g' to go first line
    • 'g' to go last line
  • man -k 'copy file'

    • sudo apt install net-tools
